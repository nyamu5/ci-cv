# TICKET 0.1 — Initialise Git Repo & Project

**Stage:** 0 — Repository Initialisation
**Goal:** A version-controlled, scaffolded Next.js project with tooling configured.

```
Set up the cv-roaster repository from scratch:

1. Create the project with pnpm:
   pnpm create next-app@latest cv-roaster --typescript --tailwind --app \
     --src-dir=false --import-alias="@/*" --no-eslint

2. cd into cv-roaster and initialise git:
   git init
   git branch -M main

3. Create a .gitignore (extend the Next.js default) to also ignore:
   .env.local
   .env*.local
   .DS_Store
   *.log
   .trigger/
   drizzle/meta/

4. Install runtime dependencies with pnpm:
   pnpm add @supabase/supabase-js @supabase/ssr drizzle-orm pg zod \
     openai pdf-parse @trigger.dev/sdk @upstash/redis @upstash/ratelimit \
     @tanstack/react-form

5. Install dev dependencies:
   pnpm add -D drizzle-kit @types/pg @types/pdf-parse @biomejs/biome \
     vitest @vitejs/plugin-react

6. Initialise Biome and create biome.json:
   pnpm biome init
   Configure: space indent, width 2, organizeImports enabled,
   recommended linter rules. Add .biomeignore for .next/, node_modules/, drizzle/

7. Initialise shadcn/ui:
   pnpm dlx shadcn@latest init
   (Style: Default, Base colour: Slate, CSS variables: yes)

   pnpm dlx shadcn@latest add button card badge tabs textarea progress \
     separator skeleton toast alert input label

8. Set up package.json scripts:
   "dev", "build", "start",
   "check": "biome check --write .",
   "typecheck": "tsc --noEmit",
   "test": "vitest run",
   "test:watch": "vitest",
   "db:generate": "drizzle-kit generate",
   "db:push": "drizzle-kit push",
   "db:studio": "drizzle-kit studio"

9. Create vitest.config.ts configured for a Node + React environment.

10. Enable TypeScript strict mode in tsconfig.json (strict: true,
    noUncheckedIndexedAccess: true).

11. Create a README.md stub with the project title and one-line description.
```

**Verification:**

```
- pnpm dev starts cleanly on localhost:3000
- pnpm typecheck passes with zero errors
- pnpm check passes (Biome runs)
- pnpm test runs (zero tests is fine at this point)
- git status shows .env.local is ignored
```

```
git add -A && git commit -m "chore: initialise project, tooling, and dependencies"
```
